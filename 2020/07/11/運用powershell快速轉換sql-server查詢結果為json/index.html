工作上時常有些資料轉換小需求 , 但一時之間又不好搞定 , 多半會透過各種亂七八糟的方式轉換 , 無聊看看 powershell 的功能裡面可以連接 sql server 操作一些這種雜亂的工作!
如果需要直接參考原生的在 sql server 上操作 json 可以看這個老外

首先需要 安裝 SQL Server PowerShell 模組

特別注意需要安裝 powershell 5.x 以上版本
