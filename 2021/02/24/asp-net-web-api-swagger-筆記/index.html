設定 logo

可以採用類似這篇老外的作法
在 Content 資料夾底下加入 logo , 接著加入自訂的 css 注意 css 要設定 Embedded Resource
在 SwaggerConfig.cs 開啟 c.InjectStylesheet 這串

在正式環境只能用 local 進入 swagger ui

所以只好搬出以前 asp.net 時代 Global.asax 這隻檔案進行全域設定
